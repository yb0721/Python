各数据类型的排序
===============
*sorted函数:所有数据类型可用*

sort方法
--------
仅list数据类型可用
列表
----
``` python
>>> d = [1, 2, 5, 3, 8, 9]
>>> d.sort()
>>> d = [1, 2, 3, 5, 8, 9]
```

元组
----
``` python
>>> d = (1, 2, 6, 5, 3,)
>>> sorted(d)
>>> d = [1, 2, 3, 5, 6]
>>> d = tuple(d)
>>> d = (1, 2, 6, 5, 3)
```

字典
----
```python
>>> d = {'a': 2, 'A': 1, 'c': 3, 'b': 2}
>>> sorted(d)
>>> d = ['A', 'a', 'b', 'c']
```

方法的使用
==========
1. strip()方法：删除开头或结尾的指定字符
``` python
>>> a = '000123000'
>>> a.strip('0')
>>> a = '123'
```
2. lstrip()方法：删除左边的指定字符
``` python 
>>> a = '000123000'
>>> a.lstrip('0')
>>> a = '123000'
```
3. replace()方法：将旧的字符串替换成新的字符串str.replace(旧的字符串，新的字符串，替换次数)
``` python
>>> a = '3.14159'
>>> a.replace('.', '', 1)
>>> a = '314159'
```
4. isnumeric()方法：检测字符串是否由数字组成，返回值为bool型
``` python
    a = '3.14159'        b = '123'
    a.isnumeric()         b.isnumeric()
    False                     True
```
5. isinstance(1，int)方法：判断所要检测对象的数据类型
``` python
>>> a = 2
>>> isinstance(a, int)
>>> True
```

------------------
